# 微信公众号排版工具 - Chrome 插件

## 安装步骤

1. 打开 Chrome，地址栏输入：`chrome://extensions/`

2. 右上角开启 **「开发者模式」**

3. 点击 **「加载已解压的扩展程序」**

4. 选择文件夹：`C:\Users\Tuo\.openclaw\workspace\wechat-formatter-chrome`

5. 插件安装完成！浏览器右上角工具栏会出现绿色图标

## 使用方法

1. 点击浏览器右上角的绿色 **W** 图标
2. 左侧粘贴或输入内容（支持 Markdown）
3. 右侧预览排版效果
4. 点击 **「复制」** 按钮
5. 公众号后台直接 Ctrl+V 粘贴

## 功能

- ✅ Markdown 实时预览
- ✅ 一键复制到剪贴板
- ✅ 支持标题、粗体、斜体、代码、引用、列表
- ✅ 公众号风格排版
